# 橘子tv安卓端
这个项目是配合橘子tv后台开发的安卓端项目，项目亮点：
+ 实现是服务器视频在安卓上播放
+ 使用了全新的Jetpack databinding绑定视图与viewmodel
+ 使用room作为持久层框架
+ 使用dagger作为依赖注入框架
+ 采用了kotlin开发语言
